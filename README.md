# MWRpy_ret

Repository for Microwave Radiometer Retrieval Derivation

## Installation

From GitHub:

```shell
git clone https://github.com/tobiasmarke/mwrpy_ret.git
cd mwrpy_ret
python3 -m venv venv
source venv/bin/activate
pip3 install --upgrade pip
pip3 install .
```

MWRpy_ret requires Python 3.10 or newer.
